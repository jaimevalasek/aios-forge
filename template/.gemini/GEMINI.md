# AIOS Forge - Gemini CLI

## Initialization
- Read `.aios-forge/config.md`
- Check `.aios-forge/context/project.context.md`
- If missing, run `aios-setup`

## Commands
Use command files in `.gemini/commands/` that point to the same agents in `.aios-forge/agents/`.
- Include `aios-ux-ui` for frontend design specification when UI quality is critical.
